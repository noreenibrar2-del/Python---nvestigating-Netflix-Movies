Investigating Netflix Movies from the 1990s

This project applies Python data analysis techniques to explore Netflix movies released during the 1990s. Using a dataset provided by DataCamp, the analysis examines the dataset to answer key questions about movie trends, genres, and characteristics from that decade.

Analysis Questions

What was the most frequent movie duration in the 1990s?
How many short action movies (under 90 minutes) were released in the 1990s?

Key Features

Data cleaning and exploration using pandas
Filtering and subsetting a Netflix movie dataset
Answering exploratory questions about 1990s movie trends

Technologies Used: Python, pandas
